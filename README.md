# Firebase function trigger notification

### Document for sending notification
```json
{
    "notification":[
        {
            "user_id":"",
            "message":"",
            "type":"",
            "created_at":"",
            "updated_at":""
        }
    ]
}
```

### Document for user store token 
```json
{
       "user":[
        {
            "user_id":"",
            "token":"",
            "created_at":"",
            "updated_at"
        }
    ]
}
```
